Privacy Policy – No Split View

Last updated: April 2026

No Split View ("the Extension") does not collect, store, transmit, or share any personal or sensitive user data.

Permissions used

  • tabs: Used solely to detect when a tab is placed into Brave's Split View and to close it automatically. No tab URLs, titles, or content are read or stored.
  • tabGroups: Used solely to detect tab grouping events triggered by Brave's Split View feature. No data is collected.

Remote code

The Extension does not load or execute any remote code. All logic is self-contained in the locally bundled background.js file.

Contact

For any questions, open an issue at github.com/Maaliselmi/nsw.
